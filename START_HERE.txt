운세 뽑기 사이트 시작 안내
===========================

1. GitHub에 올리는 방법
- 압축을 푼 뒤 안에 있는 파일과 폴더를 저장소 최상단에 모두 올리세요.
- 압축파일 자체를 올리면 안 됩니다.
- 저장소 첫 화면에 index.html, assets 폴더, src 폴더가 바로 보여야 합니다.

2. GitHub Pages 설정
- Settings > Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)
- Save
- 접속 주소: https://himnae18.github.io/pestiny/

3. 개발 화면 실행
- npm install
- npm run dev

4. 수정 후 배포 파일 갱신
- npm run build
- 루트 index.html, assets 폴더, dist 폴더가 자동 갱신됩니다.

5. 로그인 기능
- .env.example을 참고해 .env를 만들고 Supabase 정보를 입력하세요.
- supabase/schema.sql을 Supabase SQL Editor에서 실행하세요.

주의
- index.html 안에 긴 자바스크립트 코드가 직접 들어간 예전 파일은 사용하지 마세요.
- 이번 수정본은 index.html이 assets 폴더의 JS/CSS를 불러오는 정상 구조입니다.
